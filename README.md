# Seek Consciousness — GitHub-safe restored build

This version preserves the exact structure and behavior of
`seek-consciousness-kurt-photo-only.html`.

Only the embedded Base64 images and video were moved into `assets/media/`.

## Important

Upload the complete folder structure:

- `index.html`
- `assets/media/`

Do not upload only `index.html`, because the page now loads its images and
video from the media folder.

## GitHub Pages

1. Commit all files and folders.
2. Open repository **Settings → Pages**.
3. Choose the branch containing `index.html`.
4. Publish from the repository root.

The CSS and JavaScript remain inline and in their original order to preserve
all text reveals, sticky sections, scroll animations, and timing.
